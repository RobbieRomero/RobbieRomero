## 🚀 Proyectos Destacados (Producción)

### 1. Migrantes IA (SaaS de Asistencia Legal RAG)
Plataforma inteligente que asiste a extranjeros en España basándose **estrictamente en fuentes oficiales** (BOE, Ministerios). 
* **Tecnología:** Next.js, n8n, Supabase (`pgvector`), Groq Embeddings.
* **Hito:** Pipeline de ingesta documental automatizada (Scraping -> Chunking Semántico -> Base de Datos Vectorial).

### 2. RCA Master & AI Radio Automation
Un sistema completo de automatización de radio autónoma impulsada por IA.
* **Tecnología:** Liquidsoap, Icecast, n8n, ElevenLabs / Qwen, Supabase.
* **Hito:** Creación de un "Cerebro Controlador" en n8n con sistemas de *Failover* de 4 niveles, generación de jingles en tiempo real y lectura de noticias meteorológicas y RSS.

### 3. Esteban AI & AprendeIdiomas (Ecosistema PWA)
Plataforma educativa multimodal y sistema inteligente de interacciones.
* **Tecnología:** Flutter/PWA, Supabase, Integraciones API (Replicate, Pollinations.ai).
* **Hito:** Resolución de problemas complejos de estado asíncrono, recuperación RAG y pipelines de generación de imagen/video sin interrupciones.

### 4. Store IA (POS System para Restaurantes)
Sistema de Punto de Venta (POS) y E-commerce automatizado con IA.
* **Hito:** Gestión de menús dinámica, sincronización en tiempo real y automatización masiva de assets visuales mediante IA (Pollinations).
